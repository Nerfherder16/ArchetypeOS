## Content

- Event: nightly coherence probe found 10 frontend fields missing from 3 export types; a PR was cut to widen them; Guardian BLOCKed on docs and acceptance evidence.
- Root cause: Guardian's `missing-docs` and `missing-acceptance-evidence` rules are not scope-aware — they apply to any changed implementation file regardless of whether the change is a type annotation vs. new feature code.
- Decision rules for coherence-probe PRs:
  1. **Acceptance evidence**: always include a `## Acceptance Evidence` table mapping each criterion to the probe output (coherence digest path, `signal=true`, `contract_lag` entries) and schema inspection (file + line range).
  2. **Docs override** (`PR_GUARDIAN_OVERRIDE_DOCS`): justified when the only changed file is `apps/web/src/api.ts` and the change widens an `export type` to match an existing backend schema field. The backend schema (`apps/api/app/schemas.py`) is the authoritative doc; the coherence digest (`.archetype/coherence/<date>.md`) is the machine-generated change record. No new feature doc is needed.
  3. **Acceptance override** (`PR_GUARDIAN_OVERRIDE_ACCEPTANCE`): justified when the acceptance evidence table is present and the build is unrunnable due to a pre-existing failure unrelated to the type widening. Cite the pre-existing failure and its file/line.
  4. **Web-tests override** (`PR_GUARDIAN_OVERRIDE_WEB_TESTS`): justified when only `export type` annotations are added (no component logic, no hook, no render path changed). Playwright e2e tests cannot cover a type-only boundary without a working build.
  5. Each override token **must** cite this lesson (`LES-041`) so Guardian can pass with PASS_WITH_WARNINGS.
- Pre-existing build failure note (2026-07-12): `apps/web` build fails on `TS5107: moduleResolution=node10 deprecated` before reaching any changed code. This is unrelated to the coherence widening and should be fixed independently (add `"ignoreDeprecations": "6.0"` to `apps/web/tsconfig.json`, then address the deeper React type declaration gap it reveals).
