# Portfolio Knowledge Marketplace

## Purpose

The Portfolio Knowledge Marketplace makes validated engineering knowledge reusable across repositories.

It is not a public marketplace by default. It is an internal catalog of proven patterns, modules, prompts, agents, skills, MCPs, architecture packs, compliance packs, and implementation lessons.

## Examples

- Provider abstraction
- Secure upload pipeline
- Multi-tenant auth
- FastAPI scaffold
- Flutter dashboard
- HVAC sensor framework
- CPA compliance pack
- PR Guardian policy pack
- Documentation lifecycle pack

## Marketplace Entry Fields

- name
- category
- source repository
- maturity
- evidence
- dependencies
- known risks
- usage instructions
- acceptance criteria
- compatible stacks
- related decisions
- benchmark results
- owner

## Promotion Path

```text
Repository solution -> validated pattern -> reusable asset -> marketplace entry -> portfolio recommendation
```

## Principle

A solution becomes more valuable when it can teach or accelerate future systems.
