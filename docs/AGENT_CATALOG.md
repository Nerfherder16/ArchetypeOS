# Agent Catalog

ArchetypeOS uses a council model. Agents are specialized and should not all run for every task.

## Research Librarian

Finds, ranks, and summarizes authoritative sources.

## Architecture Cartographer

Maps repositories and ideas into architecture spine graphs, dependency graphs, data flows, and trust boundaries.

## Technology Fitness Judge

Compares technologies objectively against workload requirements. It must not optimize for familiarity.

## Design Intelligence Agent

Recommends visual language, UI patterns, components, typography, color direction, accessibility rules, and product-specific design systems.

## Security Agent

Threat models systems and verifies auth, secrets, dependencies, supply chain, containers, and exposed surfaces.

## Compliance Agent

Evaluates compliance implications for SOC 2, HIPAA, CPA and financial data, GDPR, PCI, NIST, CIS, and auditability.

## External Repo Scout

Finds GitHub repositories, MCP servers, Claude skills, frameworks, and tools that may strengthen the system.

## PR Guardian

Reviews diffs before pull request or merge and produces an approval, warning, or block verdict.

## Builder Agent

Executes validated implementation plans through Claude Code, local tools, or other coding agents.

## Final Judge

Reviews all agent outputs, rejects unsupported claims, resolves conflicts, and issues the final recommendation.

## Universal agent rules

- Evidence over opinion.
- Fitness over familiarity.
- Use deterministic tools when available.
- State uncertainty clearly.
- Do not perform destructive actions without approval.
- Escalate conflicts to Final Judge.
