# Engineering Observatory

## Purpose

The Engineering Observatory is the live operations view for ArchetypeOS and the engineering portfolio.

It is a NOC-style dashboard for engineering intelligence, local infrastructure, AI usage, and project health.

## Widgets

- Active agents
- Active Claude sessions
- Local LLM status
- GPU utilization
- Docker container health
- CasaOS or Portainer health
- GitHub activity
- Nightly self-learning loop status
- Research queue
- Voice inbox
- PR Guardian queue
- Cost dashboard
- Portfolio health
- Job queue
- Failed jobs
- Provider latency
- API usage

## Use Cases

- See whether local inference is healthy
- See whether agents are blocked
- See which repositories have risk
- See what the nightly loop found
- See whether PR Guardian is blocking work
- See if costs or API usage are increasing

## Principle

Engineering intelligence should be observable like production systems are observable.
