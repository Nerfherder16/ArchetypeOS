# Strategy Engine

## Purpose

The Strategy Engine helps decide what to build, improve, defer, or retire next.

It acts like an AI technical program manager and engineering strategy advisor.

## Core Questions

- Which feature should be built next?
- Which project has the highest ROI?
- Which technical debt should be paid first?
- What sequence minimizes engineering effort?
- What dependencies block future work?
- Which project should receive research or refactoring first?

## Responsibilities

- Roadmap sequencing
- ROI estimation
- Leverage analysis
- Dependency analysis
- Technical debt prioritization
- Portfolio investment guidance
- Build order recommendations
- Project comparison

## Inputs

- Roadmaps
- Repository scores
- Risks
- Technical debt
- User value
- Business value
- Engineering effort
- Dependencies
- PR Guardian results
- Nightly loop results
- Benchmark data
- Research freshness

## Outputs

- Recommended next actions
- Roadmap adjustments
- Feature sequencing
- Technical debt priority list
- Project investment recommendations
- Strategic risk report

## Principle

A good roadmap is not just a list of features. It is a sequence that improves leverage.
