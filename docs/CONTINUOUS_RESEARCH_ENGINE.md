# Continuous Research Engine

## Purpose

The Continuous Research Engine watches the engineering ecosystem for changes that may affect the portfolio.

## Sources

- GitHub releases
- GitHub trending
- MCP servers
- Claude Code ecosystem
- Docker
- Kubernetes
- AWS
- Azure
- Bedrock
- Anthropic
- OpenAI
- Gemini
- PostgreSQL
- Redis
- CVEs
- major OSS projects

## Outputs

- Better implementation alerts
- New MCP recommendations
- Security advisories
- Technology migration opportunities
- Deprecated dependency warnings
- Cost optimization ideas
- New skill suggestions

## Review Model

Research findings should become recommendations, not automatic changes.

High-impact changes require Decision Intelligence and Final Judge review.

## Cadence

- Daily lightweight scan
- Weekly portfolio digest
- Monthly technology drift review
- Immediate alerts for critical security issues
