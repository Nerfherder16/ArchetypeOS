# Engineering Memory

## Purpose

Engineering Memory is the searchable institutional memory of ArchetypeOS.

It records not only what was built, but why it was built, what failed, what was learned, and how that knowledge should influence future work.

## Artifacts

- Voice notes
- Conversations
- Research notes
- Decisions
- Code changes
- Pull requests
- Benchmarks
- Experiments
- Incidents
- Lessons learned
- Architecture graphs
- Risk reports
- Final Judge rulings

## Capabilities

- Search by decision
- Search by reasoning
- Search by failure
- Search by technology
- Search by project
- Search by reusable pattern
- Search by lesson learned

## Knowledge Lifecycle

```text
Idea -> Research -> Simulation -> Decision -> Implementation -> Validation -> Measurement -> Knowledge -> Reuse -> Evolution
```

## Principle

Engineering memory should make future decisions faster, safer, and better supported by evidence.
