# Agent Council Dashboard

## Purpose

The Agent Council Dashboard visualizes specialized agents working in parallel and gives the Final Judge a structured place to synthesize conclusions.

## Agents

- Research Librarian
- Architecture Cartographer
- Security Agent
- Compliance Agent
- Technology Fitness Judge
- Design Intelligence Agent
- External Repo Scout
- Builder Agent
- PR Guardian
- Final Judge

## Dashboard Elements

Each agent should show:

- status
- current task
- progress
- evidence gathered
- blockers
- confidence
- output summary
- escalation state

## Status States

- Waiting
- Running
- Needs Evidence
- Blocked
- Complete
- Escalated
- Rejected

## Final Judge Panel

The Final Judge should show:

- points of agreement
- points of disagreement
- unsupported claims
- evidence quality
- confidence score
- final recommendation
- required follow-up

## Principle

Do not hide disagreement. Surface it, explain it, and resolve it with evidence.
