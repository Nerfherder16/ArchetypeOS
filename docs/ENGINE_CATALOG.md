# Engine Catalog

## Engineering Evaluation Standard

Evaluates repositories for production readiness, architecture quality, security, testing, operations, and technical debt.

## Research Engine

Finds and ranks evidence from official documentation, reference implementations, security guidance, benchmarks, and community discussion.

## Knowledge Graph

Stores relationships between projects, technologies, decisions, repositories, agents, patterns, risks, and research.

## Architecture Studio

Creates editable architecture models from repositories, text, diagrams, and uploaded images.

## Architecture Spine Graph

Displays a system as a central spine of major parts with branches for subcomponents, dependencies, data flows, and risks.

## Decision Intelligence

Creates structured decision cards and ADRs with context, alternatives, evidence, tradeoffs, consequences, and approval.

## Technology Fitness Engine

Scores technologies against workload fitness rather than developer familiarity.

## Recommendation Intelligence

Identifies better implementation choices, external tools, MCPs, skills, frameworks, and migration paths.

## Design Intelligence

Recommends UI styles, components, typography, color direction, and design systems based on product purpose and user context.

## Build Intelligence

Turns validated decisions into implementation plans and hands work to Claude Code, local LLMs, or other builders.

## Evolution Engine

Revisits past decisions as tools, frameworks, models, costs, and security conditions change.

## PR Guardian

Reviews diffs before pull request or merge. Produces risk score, required fixes, and release gate status.
